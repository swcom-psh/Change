# School Seat Reassignment Program Requirements

1. **Cycle**: Monthly seat changes.
2. **Input Data**:
    - Student list.
    - Preferences: "Want to sit with" (whitelist), "Don't want to sit with" (blacklist).
    - Format: CSV file.
3. **Constraints**:
    - Physical/Vision issues: Fixed assignment to Front or Back rows.
4. **Layout**:
    - 5 Columns x 6 Rows (Total 30 seats).
5. **Platform**:
    - Web-based (HTML/JS/CSS).
    - Hosted on GitHub Pages.
